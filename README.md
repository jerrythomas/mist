# svelte-themable-ui

A set of themable UI elements.

## Usage

```bash
npm i --save-dev @jerrythomas/svelte-themable-ui@beta
```

Refer to the [example](example) for examples of all available UI elements.

## Quickstart

```bash
degit jerrythomas/svelte-charts/example my-app
cd my-app
npm i --save-dev @jerrythomas/svelte-themable-ui@beta
npm run dev
```
